# Artyrun - Landing Page

Uma landing page moderna para o Artyrun, o app que transforma seus hábitos criativos em arte generativa viva.

## 🚀 Características

- **HTML/CSS/JS Puro** - Sem dependências de build, funciona direto no navegador
- **Design Responsivo** - Perfeito em desktop, tablet e mobile
- **GitHub Pages Ready** - Hospedado gratuitamente em GitHub Pages
- **Animações Suaves** - Transições e efeitos visuais modernos
- **Performance Otimizada** - Carregamento rápido e eficiente

## 📋 Seções

1. **Hero** - Apresentação principal com call-to-action
2. **Sobre** - O que é Artyrun e seus benefícios
3. **Recursos** - Características principais do app
4. **Contato** - Formulário de inscrição beta
5. **Footer** - Links e informações adicionais

## 🎨 Customização

### Cores
Edite as variáveis CSS em `style.css`:

```css
:root {
    --primary: #6366f1;      /* Cor primária (roxo)
    --secondary: #ec4899;    /* Cor secundária (rosa)
    --accent: #f59e0b;       /* Cor destaque (ouro)
    /* ... outras cores */
}
```

### Conteúdo
Edite os textos diretamente em `index.html`.

## 🔧 Desenvolvimento Local

Simplemente abra `index.html` em um navegador. Não há necessidade de servidor ou build tools.

Ou use um servidor local:

```bash
# Com Python 3
python -m http.server 8000

# Com Node.js (http-server)
npx http-server
```

Depois acesse: http://localhost:8000
